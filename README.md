# Self-Supervised Learning for Spectrogram-Based Identification of Mobile Communications Technologies

This repository provides the **training code**, **evaluation code**, and **dataset creation code** used in the paper:  
**"Self-Supervised Learning for Spectrogram-Based Identification of Mobile Communications Technologies"**

## Training Code

### Dependencies
- `Minerva`

### Installation
```bash
pip install minerva-ml
```

## Evaluation code

### Dependencies
- `Minerva`
- `torchmetrics`

### Installation
```bash
pip install torchmetrics
```

## Dataset creating code

### Dependencies
- `scipy`
- `sigmf`
- `scikit-image`
- `numpy`
- `pandas`
- `matplotlib`

### Installation
```bash
pip install torchmetrics scipy sigmf scikit-image numpy pandas matplotlib
```
